  • git clone +Путь к репозиторию - команда для получения данных проекта из репозитория
  • git status- команда, которая показывает статус локального репозитория и отображает изменения в нем
  • git add +Имя файла - говорит git трэкать файл
  • git commit - команда фиксирующая изменения в локальном репозитории. Необходима перед отправкой данных в "Главный репозиторий"
  • git push - отправка копии локального репозитория на главный
  • git pull - получить актуальную версию проекта из репозитория
